# MockUp - Your personal AI interviewer 

A simple application for job seekers to practice for tech interviews

## Tech Stack

- Next.js
- Clerk
- Argon2
- Zod

## Phase - 1 features

- Signup and login
- AI voice chatbot which listens in realtime and answers accordingly 
- English language support
- Interview length is 5 minutes
- Assessment report after the interview 


## Phase - 2 features

- Pop up opens up where user can code and the AI can assess the code after completion or if he asks for queries without even completing it, it can help a bit
- Can be of 2 types 
    - DSA based (this phase)
    - Any VS code like online compiler for any framework (JS, TS, cpp, py, go, rust)
- Interview length up to 10-15 mins
- May even add support for hindi language
- Interview report contains graphs showing areas to work upon, Improvement graph etc
